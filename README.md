# 📊 Pesquisa de Satisfação - TudoWeb

Este programa em Python realiza uma pesquisa com clientes, coletando nome, idade e opinião sobre o atendimento (1: Excelente, 2: Bom, 3: Ruim).
Utiliza um laço `for` para repetir a pesquisa com vários entrevistados e estruturas `if/elif` para verificar as respostas. Durante a execução, 
são contabilizadas apenas as avaliações **EXCELENTE** e **RUIM**.

Ao final, o sistema exibe a quantidade total de respostas para cada uma dessas categorias.
Para testes, recomenda-se usar 10 entrevistados e depois ajustar para 50.
